What's HugPool?

HugPool is a web application created by two people in love separated by the global pandemic COVID-19.

The intention of this application is to keep a count of the hugs and kisses that you would give to that person to return them when they get back together.

Maybe it sounds a little childish, but everything is valid in order to better carry the confinement.
